## Program:
```python
#Program to find the inverse of a matrix.
#Developed by: SATHISH R
#RegisterNumber: 22006204
import numpy as np
A = np.array([[1,0,3],[-1,2,-2],[2,3,-1]])
B = np.linalg.inv(A)
print(B)
```
